## Branding swaps

- Replace `public/logo-placeholder.svg` with your real logo and update header sizing in `src/components/Header.css`.
- Replace `public/favicons/favicon.svg` and `index.html` meta tags for production URL/title/OG image.
- Update business contact details in `src/components/Footer.jsx` and `src/components/ContactCard.jsx`.

## Gallery placeholders

Currently uses Picsum placeholder images in `src/pages/Gallery.jsx`. Replace with local images in `public/gallery/` when available.
